For our MAE 3280 (Mechatronics) Course, we sought out to create a robot for a competition where it must gather as many cubes in its frame as possible within 1 minute. To win, the robot must gather more cubes than the opposing robot on the same game board. This group project involved CAD, Rapid Prototyping, Sensor input/output with an Arduino, and coding in C++ to achieve our goal.
